# 🤖 AI Assistant using Local LLM + Tools

## 📌 Overview

This project demonstrates a simple AI assistant built using a local language model and external tools.

## 🚀 Features

* Uses local LLM (GPT2 / FLAN-T5)
* Performs calculations using a custom tool
* Retrieves factual information using Wikipedia
* Handles different query types using agent logic

## 🧠 Tech Stack

* Python
* Hugging Face Transformers
* LangChain Concepts
* Wikipedia API

## ⚙️ How it works

* User input is analyzed
* Based on query:

  * Math → Custom tool
  * Factual → Wikipedia
  * General → LLM
* Returns structured output

## ⚠️ Limitations

* No real-time data everytime
* Limited creativity (local model)

## ✅ Conclusion

This project demonstrates how to build an AI agent system using LLM + tools, forming a base for advanced AI applications.
